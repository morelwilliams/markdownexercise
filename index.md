## Function to format a list of items  

```python
# Function to format a list of items
def format_list(items):
    return "\n".join(f"- {item}" for item in items)

# Example usage
shopping_list = ["Apples", "Bananas", "Cherries"]
print(format_list(shopping_list))
```
